# Why Open-Source-Hardware

Es gibt viele Gründe, warum wir  Open-Source-Hardware Trockentoiletten entwickeln wollen. Einige davon sind:

|  |  |
| ----------- | ----------- |
| Gemeinschaft | Open-Source-Hardware wird von einer Gemeinschaft von Entwicklern erstellt und verbessert, die an einem gemeinsamen Ziel arbeiten, nämlich die Schaffung von qualitativ hochwertiger Hardware. |
| Transparenz | Open-Source-Hardware ist offen und transparent. Jeder kann den Quellcode oder Bauplan einsehen und verstehen, wie die Hardware funktioniert. |
| Zusammenarbeit | Open-Source-Hardware ermöglicht es Entwicklern, zusammenzuarbeiten und ihre Fähigkeiten und Ideen zu teilen, um bessere Hardware zu erstellen. |
| Innovation | Open-Source-Hardware fördert Innovation, da Entwickler neue Ideen und Technologien leicht einbringen und testen können. |
| Freiheit | Open-Source-Hardware bietet den Nutzern Freiheit, da sie die Hardware frei nutzen, verändern und verteilen können.
| Kosten | Open-Source-Hardware ist in der Regel kostenlos, was für viele Menschen und Unternehmen von Vorteil ist.
