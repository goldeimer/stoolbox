# Por qué hardware de código abierto

Hay muchas razones por las que queremos desarrollar hardware de código abierto para inodoros secos. Algunas de ellas son:


|  |  |
| ----------- | ----------- |
| Comunidad | El hardware de código abierto es creado y mejorado por una comunidad de desarrolladores trabajando juntos hacia un objetivo común de crear hardware de alta calidad. |
| Transparencia | El hardware de código abierto es abierto y transparente. Cualquiera puede acceder y entender el código fuente o el plan de construcción y cómo funciona el hardware. |
| Colaboración | El hardware de código abierto permite a los desarrolladores colaborar y compartir sus habilidades e ideas para crear hardware mejor.|
| Innovación | El hardware de código abierto promueve la innovación ya que los desarrolladores pueden introducir y probar nuevas ideas y tecnologías fácilmente. |
| Libertad | El hardware de código abierto proporciona a los usuarios la libertad de usar, modificar y distribuir libremente el hardware.
| Costo | El hardware de código abierto puede ayudar a reducir el costo de acceso a hardware de alta calidad, ya que la producción y distribución de hardware propietario a menudo conlleva altos costos.
